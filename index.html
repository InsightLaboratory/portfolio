A comprehensive full-stack platform for geologists and mining professionals to explore, analyze, and visualize mineral exploration data in real-time. Includes interactive mapping, assay analysis, and geochemical pattern recognition.

Key Features:

  • 🗺️ Real-time drillhole locations with interactive React-Leaflet maps

  • 📈 Interactive Recharts with Au (gold) grade rankings

  • ⚡ Type-safe API integration (TypeScript + Axios)
